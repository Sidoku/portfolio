About Me

I have bad jokes, but write good code! As a programmer who loves to sketch and paint, I am focused on optimiziation and scalability while also brainstorming ideas to blend functionality with creativity. I have worked on building gameplay mechanics, Enemy AI, and game systems for 2D, 3D and VR games in Unity and Unreal engines. I have participated in multiple game jams and worked on solo projects as well througout my Masters program at Northeastern University. I am usually building games or playing them as I am always looking to expand my knowledge of differnt genres and understanding their learning curves. I am always up for a challenge and keep an optimistic outlook.

Projects

Centripetal

A Fast paced platformer game built in Unity with a focus on fast and fluid movement. This was built for a course with a focus to complete a game end - to - end. The goal my team and I set out to achieve was to make the player achieve a flow state through our gameplay mechanics and level design.

View Project

Shooter

This is a Third-Person Shooter prototype built in Unreal Engine 5. The project uses C++ and blueprints to create a extendable prototype with different weapon types, player and enemy meshes and animations while also featuring items, explosives and Enemy AI.

View Project

Frenemy Frontier

Multiplayer PvP game built in Unity with a focus on competition vs. collaboration. This game was built as part of a research study to analyze which player types choose collaboration and which choose competition. Two teams (Bounty hunters vs. Criminals) must compete against each other to win the game.

View Project

Computer Game

Action-Adventure / Role Playing Game built in Unreal Engine 5, inspired from God of War. This is a work in progress project with a functional MVP. Our main goals were to create a combat system with a level progression and test different Enemy behaviours for different enemy types.

View Project

Infernal Grace

Split screen single player game built in Unity with simultaneous control for two characters across split screen. This was built and submitted for the Global Game Jam with a theme of Duality. The game challenges the players attention and coordination to create a challenging gameplay.

View Project

Block Breaker

This is a block breaker clone built in Unity. Each level progresses with additional features added to the level design and different types of blocks. My main goal was to learn the capabilities of the engine and use modular code to allow easy configuration changes for different block health and powerups.

View Project

Laser Defender

This is a Galaga clone built in Unity. I used waypoints and coroutines amongst other concepts to setup unique enemies with varying health, speed, attack intervals and pathing. I setup projectiles for the lasers and setup a damage system with a score count.
